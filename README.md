# 🎉 Bienvenue dans mon repo de cours Ansible !

Bienvenue sur ce dépôt dédié aux cours **Ansible** de l'IMT.

Vous retrouverez ici tous les fichiers utilisés pendant les cours : playbooks, inventaires, rôles et autres ressources pédagogiques.

## 📁 Contenu du dépôt

Ce dépôt contient l'ensemble des fichiers pratiques abordés en cours, notamment :

- **Playbooks** – automatisation des tâches avec Ansible
- **Inventaires** – définition des hôtes et groupes cibles
- **Rôles** – organisation modulaire des configurations
- **Variables & templates** – personnalisation des déploiements

## 🚀 Prérequis

- [Ansible](https://docs.ansible.com/ansible/latest/installation_guide/index.html) installé sur votre machine
- Python 3.x
- Accès SSH aux machines cibles

## 📚 Ressources utiles

- [Documentation officielle Ansible](https://docs.ansible.com/)
- [Ansible Galaxy](https://galaxy.ansible.com/)

## 🤝 Contribution

Ce dépôt est destiné à un usage pédagogique. N'hésitez pas à ouvrir une _issue_ ou une _pull request_ pour toute suggestion ou correction.
