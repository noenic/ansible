# Bienvenue dans mon repo de cours Ansible !

Bienvenue sur ce dépôt dédié aux cours **Ansible** de l'IMT.

Vous retrouverez ici tous les fichiers utilisés pendant les cours : playbooks, inventaires, rôles et autres ressources pédagogiques.

## Contenu du dépôt

Des petits rapports sur les différentes parties du [cours](https://formations.microlinux.fr/ansible/a-propos/) avec des captures d'écran et et parfois des explications, et quand c'est nécessaire des fichiers de configuration ou des playbooks.


>Afin de ne pas trop alourdir le repo, nous nous occuperons uniquement des exercices dit "A vous de jouer"
